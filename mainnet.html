Monad Mainnet

Monad mainnet launched in November 2025. It is a production-grade, high-performance EVM blockchain with Chain ID 143 and MON as the native gas token, delivering 10,000+ TPS with 1-second block times.

Migrating from Ethereum

Monad is 100% EVM-compatible. Moving an existing Ethereum application to Monad requires only changing the RPC URL and chain ID. The following summarizes what stays the same and what to update.

What stays the same
  • ✓ Solidity smart contracts (no changes needed)
  • ✓ ethers.js, viem, web3.js
  • ✓ Hardhat, Foundry, Remix
  • ✓ MetaMask and EVM-compatible wallets
  • ✓ OpenZeppelin libraries
  • ✓ EIP-1559 gas pricing
What to update
  • ⚠ RPC URL → Dwellir Monad endpoint
  • ⚠ Chain ID → 143
  • ⚠ Gas token → MON (not ETH)
  • ⚠ Block explorer → monadexplorer.com
  • ⚠ Mempool assumptions → local mempool design
