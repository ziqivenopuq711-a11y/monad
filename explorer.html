Monad Explorer

The Monad Block Explorer lets you inspect transactions, blocks, addresses, tokens, and smart contracts on the Monad blockchain in real time. Available for both mainnet and testnet.

How to Use the Explorer

Search Transactions

Paste a transaction hash (0x…) into the search bar to view the full transaction detail: sender, receiver, value transferred, gas used, input data, event logs emitted, and block confirmation. Given Monad's local mempool design, a submitted transaction will appear in the explorer once it is included in a block, typically within 1–2 seconds of submission.

Search Blocks

Enter a block number or block hash to view the full block: timestamp, validator, gas used, transaction count, state root, and all transactions included in that block. Monad produces a new block approximately every 1 second, so blocks accumulate rapidly. You can browse the latest blocks in real time from the explorer home page.

Search Addresses

Enter a wallet or contract address to view its MON balance, full transaction history, token holdings (ERC-20, ERC-721, ERC-1155), and — for smart contracts — the ABI, source code, and read/write interface. Verified contracts on Monad Explorer are marked with a checkmark and expose their full source code for transparency.

Explorer Features

Contract Verification

Submit contract source code to verify it on-chain. Supports Solidity and Vyper with standard JSON input format. Verified contracts expose their ABI and source code publicly.

Token Tracking

View ERC-20, ERC-721, and ERC-1155 token transfers, holder lists, and total supply for any token contract deployed on Monad. Filter token transfers by address.

Read/Write Interface

Interact with verified contracts directly in the browser — read state variables and call write functions with your MetaMask wallet connected to Monad.
