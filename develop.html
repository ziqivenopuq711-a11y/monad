Developer Guide

Build on Monad using your existing Ethereum tooling. Monad is EVM-compatible — any library, framework, or tool that works on Ethereum works on Monad without modification.

Connecting with ethers.js

Use ethers.js v6 (or v5) to connect to Monad. Replace YOUR_API_KEY with your Dwellir API key obtained from the Dwellir dashboard. 

import { ethers } from "ethers";

const provider = new ethers.JsonRpcProvider(
  "https://monad-mainnet.dwellir.com/YOUR_API_KEY"
);

const network = await provider.getNetwork();
console.log("Chain ID:", network.chainId); // 143n

const blockNumber = await provider.getBlockNumber();
console.log("Latest block:", blockNumber);

Connecting with viem

Viem is a modern, type-safe Ethereum library. Define the Monad chain and create a public client pointing at your Dwellir RPC endpoint. Viem's TypeScript types provide full autocompletion for all Monad-compatible RPC methods. 

import { createPublicClient, http, defineChain } from "viem";

const monad = defineChain({
  id: 143,
  name: "Monad",
  nativeCurrency: { name: "MON", symbol: "MON", decimals: 18 },
  rpcUrls: {
    default: {
      http: ["https://monad-mainnet.dwellir.com/YOUR_API_KEY"],
    },
  },
  blockExplorers: {
    default: { name: "Monad Explorer", url: "https://monadexplorer.com" },
  },
});

const client = createPublicClient({ chain: monad, transport: http() });
const block = await client.getBlockNumber();
console.log("Block:", block);

Smart Contract Deployment with Hardhat

Add Monad as a network in your hardhat.config.js. All Hardhat features — compilation, testing, deployment scripts, gas reporting — work unchanged on Monad. 

// hardhat.config.js
require("@nomicfoundation/hardhat-toolbox");

module.exports = {
  solidity: "0.8.24",
  networks: {
    monad: {
      url: "https://monad-mainnet.dwellir.com/YOUR_API_KEY",
      chainId: 143,
      accounts: [process.env.PRIVATE_KEY],
    },
    monadTestnet: {
      url: "https://monad-testnet.dwellir.com/YOUR_API_KEY",
      chainId: 10143,
      accounts: [process.env.PRIVATE_KEY],
    },
  },
};

Deploy with:

npx hardhat run scripts/deploy.js --network monad

Development Tools

Hardhat

The most popular Ethereum development framework. Supports contract testing, deployment scripts, gas reporting, and a local Hardhat Network for rapid iteration. Works with Monad out of the box by configuring the network URL and chain ID.

Foundry

A blazing-fast Rust-based toolkit for Ethereum development. Use forge for testing, cast for RPC calls, and anvil for local node simulation. Configure with --rpc-url pointing to your Dwellir endpoint.

Remix IDE

Browser-based IDE at remix.ethereum.org. Use the "Injected Provider" environment with MetaMask configured for Monad, or select "Custom External HTTP Provider" and enter your Dwellir endpoint directly.

Foundry — cast example

Query Monad directly from the command line:

cast block-number \
  --rpc-url https://monad-mainnet.dwellir.com/YOUR_API_KEY

cast balance 0xYourAddress \
  --rpc-url https://monad-mainnet.dwellir.com/YOUR_API_KEY
