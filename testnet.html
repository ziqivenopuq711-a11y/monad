Monad Testnet

The Monad testnet (Chain ID 10143) is a public test network for developers to deploy and test smart contracts without spending real MON tokens. It mirrors mainnet behavior and supports the same tooling.

Faucet — Getting Testnet MON

To interact with the Monad testnet you need testnet MON tokens for gas fees. Testnet MON has no monetary value and is used only for development and testing. The official Monad faucet distributes testnet MON to developers upon request, enabling you to deploy contracts and test transactions without any financial cost.

How to get testnet MON:
  1. Visit the official Monad faucet at faucet.monad.xyz
  2. Connect your wallet or enter your Ethereum-compatible address
  3. Complete any required verification step
  4. Receive testnet MON within seconds to your wallet

Connect Wallet to Testnet

Add the Monad testnet to MetaMask using the parameters below. You can also programmatically add the network from your dApp to streamline the user onboarding experience.

MetaMask — Add Network Manually
  1. MetaMask → Settings → Networks → Add a network → Add manually
  2. Network Name: Monad Testnet
  3. RPC URL: your Dwellir testnet endpoint
  4. Chain ID: 10143
  5. Symbol: MON
  6. Explorer: https://testnet.monadexplorer.com

Or add the network programmatically from your dApp:

await window.ethereum.request({
  method: "wallet_addEthereumChain",
  params: [{
    chainId: "0x27AF",  // 10143 in hex
    chainName: "Monad Testnet",
    nativeCurrency: { name: "MON", symbol: "MON", decimals: 18 },
    rpcUrls: ["https://monad-testnet.dwellir.com/YOUR_API_KEY"],
    blockExplorerUrls: ["https://testnet.monadexplorer.com"],
  }],
});
